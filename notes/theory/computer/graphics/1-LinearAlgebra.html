线性代数基础

向量(Vector)

图形学的基本研究对象，有点的位置、法向等。

而从本质上来说，均为向量。

那么，什么是向量？

在中学数学或者物理学中，我们认为，既有大小又有方向的量，即为向量，其运算满足平行四边形法则。

而在线性代数中，我们认为，在向量空间（线性空间）中的元素，即为向量，其运算满足公理化定义。

向量空间(Vector Space)

向量空间

在数域 FFF 上的向量空间，可以表示为带有向量加法和标量乘法的非空集合 VVV。

其中，各类运算如下：

  • 向量加法结合律：u+(v+w)=(u+v)+w\boldsymbol{u} + (\boldsymbol{v} + \boldsymbol{w}) = (\boldsymbol{u} + \boldsymbol{v}) + \boldsymbol{w}u+(v+w)=(u+v)+w
  • 向量加法交换律：u+v=v+u\boldsymbol{u} + \boldsymbol{v} = \boldsymbol{v} + \boldsymbol{u}u+v=v+u
  • 向量加法单位元：v+0=0\boldsymbol{v} + \boldsymbol{0} = \boldsymbol{0}v+0=0
  • 向量加法逆元：v+−v=0\boldsymbol{v} + \boldsymbol{-v} = 0v+−v=0
  • 标量乘法与数域乘法的结合律：a(bv)=(ab)va(b\boldsymbol{v}) = (ab)\boldsymbol{v}a(bv)=(ab)v
  • 标量乘法单位元：1v=v1\boldsymbol{v}=\boldsymbol{v}1v=v
  • 标量乘法对向量加法的分配律：a(u+v)=au+ava(\boldsymbol{u} + \boldsymbol{v}) = a \boldsymbol{u} + a \boldsymbol{v}a(u+v)=au+av
  • 标量乘法对数域加法的分配律：(a+b)v=av+bv(a+b) \boldsymbol{v} = a \boldsymbol{v} + b \boldsymbol{v}(a+b)v=av+bv

数域

在图形学应用中，数域 FFF 一般取为实数域 R\mathbb{R}R，其他选择包括复数域 C\mathbb{C}C 等。

数域 FFF 中存在不全为 0 的一组数 a1,a2,⋯ ,ana_1, a_2, \cdots, a_na1​,a2​,⋯,an​ 使得

  • a1u1+a2u2+⋯+anun=0a_1\boldsymbol{u_1}+a_2\boldsymbol{u_2}+\cdots+a_n\boldsymbol{u_n} = 0a1​u1​+a2​u2​+⋯+an​un​=0

向量空间的维度

空间内能找出的线性无关的向量的个数的最大值，记为 dim⁡V\dim VdimV

线性组合的系数 (coefficients) 被称为该矢量在这组基下的坐标 (coordinates)。

图形学研究的维度

图形学研究的维度包含低维向量和高维向量。

低维向量和向量空间（2D~4D）的例子包括：

  • 物理空间：Mesh、曲线、点云的坐标及导数
    • 欧几里得空间 (x,y,z)(x,y,z)(x,y,z)
    • 闵可夫斯基空间 (x,y,z,ict)(x,y,z,ict)(x,y,z,ict)
  • 颜色空间：RGB, CMYK

高维向量和向量空间的例子包括：

  • 灰度数字图像上所有像素值组成的向量
    • 如 1920 × 1080 的灰度数字图像维度达到 200 万
  • 二维或三维图形的所有自由度组成的向量
    • SIGGRAPH 水体模拟求解的向量维度一般在 10610^6106 至 10710^7107

线性映射

线性映射，即将一个空间中的向量映射到另一个空间中，定义如下：

线性映射

f:V→Wf:V \rightarrow Wf:V→W，其中：

  • VVV，WWW均为向量空间
  • f(u+v)=f(u)+f(v)f(\boldsymbol{u} + \boldsymbol{v}) = f(\boldsymbol{u}) + f(\boldsymbol{v})f(u+v)=f(u)+f(v)
  • f(av)=αf(v)f(a\boldsymbol{v}) = \alpha f(\boldsymbol{v})f(av)=αf(v)
  • 推论：
    • f(0)=0f(\boldsymbol{0}) = \boldsymbol{0}f(0)=0
    • f(au+bv)=af(u)+bf(v)f(a\boldsymbol{u}+b\boldsymbol{v}) = af(\boldsymbol{u})+bf(\boldsymbol{v})f(au+bv)=af(u)+bf(v)

在低维空间的线性映射中，要注意的是：

  1. 缩放、旋转是线性映射
  2. 平移不是线性映射
  3. 仿射变换 (affine transformation) = 缩放、旋转 + 平移

矩阵

什么是矩阵？

我们不认为矩阵是一种二维数字阵列，而认为矩阵是对线性映射的一种表示。可以参考3B1B的动画: 线性代数的本质open in new window

矩阵运算的意义

矩阵与向量的乘法，可以理解成向量在新的空间（经过矩阵对应的线性变换后的空间）里的坐标。

矩阵之间的乘法，可以理解为对空间的多次相继变换的合成。

方阵，即 n×nn \times nn×n 的矩阵，暗示变换前与变换后的空间有相同的维度

行列式

在 nnn 阶方阵中选 nnn 个元素使得每行每列各有一个元素被选出，求其乘积，再将不同选择方案的乘积乘以正负 1（由选择方法的奇偶性决定）加和

意义在于，矩阵对应的线性变换对空间的拉伸程度的度量，即物体经过变换前后的体积比。

计算方法：

  • 低维矩阵，nnn 条主对角线分别计算乘积的和，减去 nnn 条次对角线分别计算乘积的和。
  • 高维矩阵：高斯消元后取对角线元素的积。

性质：

  • 转置不变；
  • 交换行（列）取反
